# UI Automation Testing of Inside Track using Cypress 
This is the test automation suite for [https://lizzip.net/insidetrack](https://lizzip.net/insidetrack) created using Cypress  
  
    
## Installation 
This test pack requires NodeJS and Cypress to be installed 
- NodeJS can be installed from [https://nodejs.org/](https://nodejs.org/)
- Once NodeJS is installed Cypress can be installed with `npm install cypress`  
  

## Usage
The tests can either be run using the default Cypress commands `npx cypress open` and `npx cypress run` or with the following commands: 
- `npm run cypress` to open the Cypress UI
- `npm run tests` to run all tests on the command line 
- `npm run fixture-tests` to run only the tests using fixtures on the command line
